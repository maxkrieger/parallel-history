A browser extension that replaces your new tab page with history. A collaboration with the [Edvo](https://edvo.com) team.

Parallel History .

## Development

Requires node, yarn, and Firefox.

### Setup

```
yarn
```

### Running

```
yarn start
```

### Building (for production)

```
yarn run build
```
